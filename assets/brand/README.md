# Brand assets

The **ED cell monogram** — the letters E and D built from spreadsheet
cells, with one highlighted "selected" cell. Designed July 2026 for the
Excel-era brand.

| File | Use |
|------|-----|
| `ed-mark.svg` | Green mark on transparent — light backgrounds, documents |
| `ed-mark-white.svg` | White mark on transparent — dark backgrounds, video overlays |
| `ed-mark-tile.svg` | White mark on a green rounded tile — social avatars, app icons |
| `avatar.svg` | 800×800 avatar with subtle grid — YouTube / LinkedIn profile |

**Colours**

| Token | Hex |
|-------|-----|
| Brand green | `#0E7A4F` |
| Highlight cell | `#6FC39A` (on light) / `#A8DEC5` (on green/dark) |
| Ink | `#1C2823` |
| Paper | `#FBFAF7` |

To export PNGs at any size, open an SVG in a browser and screenshot, or
use any SVG→PNG converter — the artwork is resolution-independent.

The previous network-era brand (cyan ED mark, intro motion files) is
preserved on the `archive/ccna-site-2026-07` branch.
